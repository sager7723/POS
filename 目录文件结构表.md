PoS/
├─ pos/                      # 核心协议实现（所有逻辑代码）
│  ├─ crypto/               # 密码学模块（当前为 mock 实现）
│  ├─ protocol/             # 协议流程（按专利步骤拆分）
│  ├─ models/               # 数据结构定义（各阶段对象）
│
├─ stages/                  # 各阶段独立入口（单阶段性能测试）
│
├─ tests/                   # 单元测试
│
├─ main_avg_time_and_winning_rate.py         #3候选者时的时间统计  
├─ main_performance_curve.py    # 性能实验（规模扩展曲线）


各子目录的结构表：

crypto/
├─ setup.py
├─ fhe.py
├─ secret_sharing.py
├─ commitment.py
├─ dkg.py
├─ random_seed.py
├─ key_homomorphic_prf.py
├─ proofs.py
├─ ticket.py

| 文件                     | 作用                        |
| ---------------------- | ------------------------- |
| setup.py               | 实现步骤0：生成公共参数（pp）          |
| fhe.py                 | 门限全同态加密接口（加密/计算/解密）       |
| secret_sharing.py      | 秘密分享与恢复（Shamir mock）      |
| commitment.py          | Pedersen承诺（隐藏质押）          |
| dkg.py                 | 分布式密钥生成（PK / 解密分片）        |
| random_seed.py         | 分布式随机种子生成                 |
| key_homomorphic_prf.py | 密钥同态PRF（生成伪随机数分片）         |
| proofs.py              | ZK证明分片生成（cut-and-choose用） |
| ticket.py              | 票根生成、哈希拆分与加密              |

models/
├─ common.py
├─ stage2.py
├─ stage3.py
├─ stage4.py
├─ stage5.py

| 文件        | 作用                     |
| --------- | ---------------------- |
| common.py | 公共参数（pp）定义             |
| stage2.py | 参与者、承诺、密钥分片等结构         |
| stage3.py | 参选消息（CandidateMessage） |
| stage4.py | 验证/抽签阶段结果结构            |
| stage5.py | 揭示阶段结果（winner等）        |

protocol/
├─ initialization.py
├─ preparation.py
├─ candidacy.py
├─ election.py
├─ reveal.py

| 文件                | 对应专利阶段       | 作用             |
| ----------------- | ------------ | -------------- |
| initialization.py | 阶段1（步骤0）     | 系统初始化          |
| preparation.py    | 阶段2（步骤1-3）   | 承诺、DKG、随机种子    |
| candidacy.py      | 阶段3（步骤4-8）   | 构造参选消息         |
| election.py       | 阶段4（步骤9-18）  | 验证 + 抽签        |
| reveal.py         | 阶段5（步骤19-23） | 揭示 + 验证 winner |

stages/
├─ stage1_main.py
├─ stage2_main.py
├─ stage3_main.py
├─ stage4_main.py
├─ stage5_main.py

| 文件             | 作用         |
| -------------- | ---------- |
| stage1_main.py | 单独测试初始化时间  |
| stage2_main.py | 单独测试参选前准备  |
| stage3_main.py | 单独测试参选消息构造 |
| stage4_main.py | 单独测试验证与抽签  |
| stage5_main.py | 单独测试揭示阶段   |

tests/
├─ test_stage1.py
├─ test_stage2.py
├─ test_stage3.py
├─ test_stage4.py
├─ test_stage5.py
